# wtrec
A Cli for to record body weight in JSON.

### Installation
```sh
git clone https://github.com/mt-coff/wtrec
cd wtrec
npm install -g
```

### Usage
```sh
wtrec init     # create weight.json
wtrec add 100  # record 100 in JSON
wtrec reset    # reset weight.json
```
